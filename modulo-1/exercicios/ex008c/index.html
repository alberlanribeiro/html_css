Revisando alguns conceitos de formatação de textos

Texto em negrito

Para pôr um texto em negrito utilize a tag strong, strong=forte

Texto em itálico

Para pôr um texto em itálico utilize a tag em, em=emphasis=ênfase

Texto marcado

Para marcar um texto utilize a tag mark, a mudança de cor é feita pelo CSS

Texto deletado

Para inserir um texto deletado utilize a tag del

Texto sublinhado ou inserido

Para pôr um texto em sublinhado utilize a tag ins, ins=insert=inserir

Texto sobrescrito

Para pôr um textosobrescrito utilize a tag sup, sup=superscript

Texto subscrito

Para pôr um textosubscrito utilize a tag sub, sub=subscribed
